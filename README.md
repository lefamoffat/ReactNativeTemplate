# ReactNativeTemplate

This app aims to be a basic bootstrap application, that can then easily be extended based on requirements.


# Running the app
To start the server:
`yarn start `

To start on an emulator:

Android - (make sure the emulator is running already!):
`yarn android`

IOS:
`yarn ios`
